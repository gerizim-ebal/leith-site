# leith-site

One-page static website for a NZ-based contractor offering Agile & Scrum Coaching, Clifton Strengths Coaching, and Change Management services.

## Files

| File | Purpose |
|---|---|
| `index.html` | Page structure and content |
| `style.css` | Styling — mobile-first, responsive |
| `script.js` | Mobile nav toggle and footer year |

## Deployment

Hosted on GitHub Pages at `https://gerizim-ebal.github.io/leith-site/`

## Making changes

Edit files locally, then:

```
git add .
git commit -m "describe your change"
git push
```

The site updates automatically within ~60 seconds of pushing.
